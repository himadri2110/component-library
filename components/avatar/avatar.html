Avatar

An avatar is a graphical representation of a user or the user's character or persona.
Circle Avatar
A circle avatar is the default style of the avatar that is used to represent user profile or account. It has 4 sizes: lg, md, sm, xs.
Square Avatar
A square avatar is the secondary style of the avatar, which can be generated by adding a avatar-sq class to the default avatar.
