Image

An image can be reponsive, which fits in all screen sizes. Images can also be round and square.
Responsive Image
A responsive image will adapt to the width of it's parent container. Resize the brower window to see the image behave responsive. You can set custom width of the resp-img class, and the img will be responsive wrt to it.
Round Image
A round image is circular in shape, and has the default size of 200 X 200px.
Square Image
A square image has equal width and height of 200px.
