Button

A button is a clickable element used to perform some action on a web page. They are an integral of any dynamic web page.
Solid Buttons
They are various types of solid buttons:
Outline Button
A button with an outline, which can be used for secondary actions.
Link Button
A button with a link, that can be used to navigate to different pages.
Link text
Icon Button
A button with a icon imparts more knowledge about the button action.
Floating Action Button
A floating action button is used to perform the primary or the most common action on a web page. They can be found on the bottom right corner of the page.
